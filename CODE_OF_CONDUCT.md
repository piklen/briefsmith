# Code of Conduct

## Our Standard

We want this project to be useful, direct, and technically rigorous.

Participants are expected to:

- communicate with respect
- criticize ideas with technical reasons, not personal attacks
- stay honest about evidence, uncertainty, and verification
- avoid harassment, discrimination, or abusive behavior

## Unacceptable Behavior

Examples include:

- personal attacks or insults
- harassment or intimidation
- discriminatory language or conduct
- publishing private information without permission
- knowingly misrepresenting verification results or security impact

## Enforcement

Project maintainers may remove, edit, or reject contributions and discussions that violate this code of conduct.

Repeated or severe violations may result in temporary or permanent exclusion from the project space.
